# menu.beersheep.rs

## deployment

- update taps via bot
- trigger deployment
- wait for a few seconds
- check the website for changes

## development

### prerequisites

- install [nvm](https://www.nvmnode.com/guide/installation.html)
- `nvm install && nvm use`
- `npm install`

### build and debug

- Add keys for db to `.env`
- `npm run serve`
- visit `http://bs-local.com:8000/`
